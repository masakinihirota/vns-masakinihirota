# 更新情報

直近の主な更新内容です。

## 2026年2月15日

- **認証・データベース基盤の大規模リファクタリング**:
  - 認証システムを Supabase Auth から **Auth.js (NextAuth.js)** に完全に移行しました。これにより、より柔軟でセキュアな認証が可能になりました。
  - データベース操作を **Drizzle ORM** に統合しました。型安全性が向上し、将来の拡張やメンテナンスが容易になりました。
  - プロフィール、グループ、イベント、マーケット、通知等、全てのデータアクセス層を最新の技術基盤に刷新しました。
- **匿名ログインの一時停止**: 認証基盤の移行に伴い、匿名ログイン機能をメンテナンスのため一時停止しました。

## 2026年2月10日

- **グループ機能の強化**: グループ作成時のリーダー選択フローを改善し、より分かりやすくしました。
- **データベースの最適化**: グループやプロフィールのデータ処理を高速化し、動作を安定させました。
- **セキュリティ強化**: データの取り扱いに関するルールを IPA（情報処理推進機構）の基準に合わせて強化しました。

## 2026年2月9日

- **「Good Life」タブの追加**: 人生の質を高めるためのコンテンツを追加しました。
- **翻訳機能（日英対応）**: 「Good Life」タブ内のコンテンツを英語と日本語で切り替えられるようにしました。

## 2026年2月8日

- **オンボーディング（導入ガイド）の改善**: 初めて使う方が迷わないよう、初期設定のステップを簡略化・最適化しました。
- **ダークモードの導入**: 目に優しい「エレガントなダークモード」と、明るい「グラスモーフィズム・ライトモード」を切り替えられるようにしました。
