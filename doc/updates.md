# 🔄 更新情報

## 最新のアップデート (2026-02-09)

### ✨ Communityダッシュボードのリニューアル

コミュニティ活動の核となる「/groups」ページを全面的にアップデートしました！

- **統合UI**: 国、グループ、マーケット、イベント、通知を一画面でシームレスに切り替えられるようになりました。
- **ポイント経済（エスクロー）**: マーケットでの取引時にシステムがポイントを一時預かる仕組みを導入し、安心・安全な取引が可能になりました。
- **建国システム**: ポイントを消費して自分の「国」を興すことができるようになりました。
- **イベント管理**: 簡単にイベントに参加・キャンセルできる直感的な操作感を実現しました。

## 2026-02-08

### ✨ デザインリニューアル: 作品リスト

「作品リスト」画面のデザインを一新しました！

- **Glassmorphism（すりガラス）デザイン**:
  - 透明感のある美しいカードデザインを採用しました。
  - 背景がうっすら透ける、モダンで軽やかな見た目になりました。

- **エレガントなダークモード**:
  - 夜間でも目に優しい、上質で落ち着いたダークテーマに対応しました。
  - 従来の単なる黒反転ではなく、深みのある色合いで没入感を高めます。

- **操作性の向上**:
  - 評価（Tier）の変更がよりスムーズになりました。
  - フィルタリングや並び替えの使い勝手を改善しました。

### 🔧 機能改善

- **バックエンド連携の強化**: 作品データの保存・読み込みがより高速・確実になりました。
- **UUID対応**: システム内部のID管理を最新化し、安定性を向上させました。
- **開発ダッシュボード改善**: 開発用ダッシュボードのリンク網羅性を向上させ、すべてのページへのアクセスを容易にしました。
