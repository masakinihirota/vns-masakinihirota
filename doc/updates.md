# 更新情報

直近の主な更新内容です。

## 2026年2月15日 (Phase 4/5 強化)

- **マッチングロジックの設計 (Phase 5)**:
  - タグマッチング(Base)とベクトル類似度(Boost)を組み合わせたハイブリッドアルゴリズム(`doc/architecture/matching_logic.md`)を策定しました。
  - 「価値観ベースのマッチング」をより科学的かつ直感的に行うための基盤を構築しました。
- **経済圏UIの刷新 (Phase 4)**:
  - `Marketplace` のUIにGlassmorphismを採用。磨りガラスのような背景と流れるようなグラデーションで、プレミアムな取引体験を実現しました。
  - `Dev Dashboard` を更新し、Phase 4/5の新機能ステータスをバッジで可視化しました。
- **認証・データベース基盤の大規模リファクタリング**:
  - 認証システムを Supabase Auth から **Auth.js (NextAuth.js)** に完全に移行しました。
  - データベース操作を **Drizzle ORM** に統合し、型安全性と安定性を向上させました。
- **コード品質の向上**:
  - `oxlint` による静的解析を実施。未使用変数のクリーンアップやビルドの安定化を行いました。


## 2026年2月10日

- **グループ機能の強化**: グループ作成時のリーダー選択フローを改善し、より分かりやすくしました。
- **データベースの最適化**: グループやプロフィールのデータ処理を高速化し、動作を安定させました。
- **セキュリティ強化**: データの取り扱いに関するルールを IPA（情報処理推進機構）の基準に合わせて強化しました。

- **翻訳機能（日英対応）**: 「Good Life」タブ内のコンテンツを英語と日本語で切り替えられるようにしました。

- **オンボーディング（導入ガイド）の改善**: 初めて使う方が迷わないよう、初期設定のステップを簡略化・最適化しました。
