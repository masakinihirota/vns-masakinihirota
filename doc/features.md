# 機能一覧と実装状況

VNS masakinihirota の現在の機能実装状況と、将来の計画一覧です。

システム用語は使わず、**「何ができるか」** という視点で記載しています。

## ✅ 実装済み機能（すぐ使えます）

### 👤 アカウント・プロフィール

| 機能                     | 説明                                                                                   | 状態        |
| :----------------------- | :------------------------------------------------------------------------------------- | :---------- |
| **ログイン/登録**        | Googleアカウント、GitHubアカウント、または匿名でログインできます。                     | 🟢 利用可能 |
| **ルートアカウント**     | あなたの活動の基盤となるメインアカウントです。ポイントやレベルを管理します。           | 🟢 利用可能 |
| **ユーザープロフィール** | 「仕事用」「趣味用」など、目的に合わせて複数のプロフィール（仮面）を使い分けられます。 | 🟢 利用可能 |
| **名刺表示**             | プロフィールをデジタル名刺として表示・共有できます。                                   | 🟢 利用可能 |

### 💖 価値観・作品

| 機能                 | 説明                                                                   | 状態        |
| :------------------- | :--------------------------------------------------------------------- | :---------- |
| **価値観登録**       | 大切にしている価値観や考え方を登録し、マッチングの基準にします。       | 🟢 利用可能 |
| **作品登録**         | 好きな映画、本、ゲームなどを登録し、ティア（Tier）評価を付けられます。 | 🟢 利用可能 |
| **作品リスト**       | 登録した作品を一覧表示し、並び替えやフィルタリングができます。         | 🟢 利用可能 |
| **マンダラチャート** | 9x9のマスを使って目標達成のための行動を整理・可視化できます。          | 🟢 利用可能 |

### 🤝 つながり・組織

| 機能                   | 説明                                                                                       | 状態        |
| :--------------------- | :----------------------------------------------------------------------------------------- | :---------- |
| **自動マッチング**     | 価値観が似ている人を毎日自動で紹介します。                                                 | 🟢 利用可能 |
| **手動マッチング**     | 条件を指定して、気の合う仲間を探せます。                                                   | 🟢 利用可能 |
| **組織（グループ）**   | 同じ目的を持つ仲間と少人数のグループを作れます。リーダーやメンバーとしての活動が可能です。 | 🟢 利用可能 |
| **国（コミュニティ）** | 複数のグループが集まる大規模なコミュニティ「国」に参加できます。                           | 🟢 利用可能 |
