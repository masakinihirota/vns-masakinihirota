# VNS masakinihirota 機能一覧 (実装ロードマップ)

本ドキュメントは、VNS masakinihirota の現在の実装状況と、今後の実装予定を対比させたガイドです。
実際のアプリの実装（コード/UI）と、要件定義書に基づいて「何ができて、何がこれからか」を明記しています。

## 1. ユーザー基盤・認証

ユーザーがアプリに参加するための基礎機能です。

| 機能名                 | ステータス   | 内容                                                             |
| :--------------------- | :----------- | :--------------------------------------------------------------- |
| **匿名体験モード**     | `[実装済み]` | ログインなしでアプリの雰囲気を体験できます。                     |
| **外部アカウント連携** | `[実装済み]` | Google/GitHubアカウントで安全にログインできます。                |
| **ルートアカウント**   | `[実装済み]` | あなたの本体となるアカウントです。ポイントやレベルが紐付きます。 |
| **オンボーディング**   | `[実装済み]` | 初めて使う方向けの初期設定ガイドが用意されています。             |

## 2. ユーザープロフィール (千の仮面)

「目的ごとに自分を使い分ける」VNSの核となる機能です。

| 機能名                     | ステータス   | 内容                                                               |
| :------------------------- | :----------- | :----------------------------------------------------------------- |
| **千の仮面 (複数プロフ)**  | `[実装済み]` | 仕事用、趣味用など複数のプロフィールを使い分けられます。           |
| **プロフィール切り替え**   | `[実装済み]` | 画面上部のセレクターで、瞬時に仮面を切り替えられます。             |
| **名刺表示モード**         | `[実装済み]` | 仕事の相手に渡すための、整理されたプロフィール表示です。           |
| **パージ機能(紐付け解除)** | `[計画中]`   | 特定のプロフィールを本体から完全に切り離す究極のプライバシー保護。 |

## 3. 作品登録・評価

あなたの「好き」を可視化し、マッチングの材料にします。

| 機能名                       | ステータス   | 内容                                                           |
| :--------------------------- | :----------- | :------------------------------------------------------------- |
| **作品カタログ登録**         | `[実装済み]` | 好きなアニメ、本、ゲームなどの作品を登録できます。             |
| **ティア評価 (Tier1-3)**     | `[実装済み]` | その作品がどれくらい好きかを3段階で表現できます。              |
| **継続評価・人気ランキング** | `[実装済み]` | 他のユーザーにも人気の作品や、継続して楽しんでいる作品の確認。 |
| **コンテンツ連鎖(チェーン)** | `[計画中]`   | 作品と関連する楽曲や動画、解説を繋ぎ合わせる機能。             |

## 4. マッチング

価値観の合う相手を見つける機能です。

| 機能名                     | ステータス       | 内容                                                   |
| :------------------------- | :--------------- | :----------------------------------------------------- |
| **自動マッチング (AI)**    | `[実装済み(UI)]` | AIが価値観を分析し、最適な相手を提案する出発点。       |
| **手動マッチング**         | `[実装済み(UI)]` | 特定の条件で相手をじっくり探すモード。                 |
| **マッチングアルゴリズム** | `[開発中]`       | 価値観の類似性を高度な点数で算出するロジックの強化中。 |

## 5. 価値観・成長・コミュニティ

より深い交流と、自分自身の成長を記録する機能です。

| 機能名                  | ステータス   | 内容                                                     |
| :---------------------- | :----------- | :------------------------------------------------------- |
| **価値観アンケート**    | `[実装済み]` | 質問に答えることで、あなたの考え方をシステムに伝えます。 |
| **マンダラチャート**    | `[実装済み]` | 目標達成のための強力な可視化ツール。                     |
| **組織 (ボトムアップ)** | `[計画中]`   | 価値観の合う少人数でのグループ作成・管理機能。           |
| **国 (トップダウン)**   | `[計画中]`   | 多数の組織が集まる大規模なコミュニティ空間。             |
| **選挙・調停制度**      | `[計画中]`   | コミュニティの方針を民主的に決める仕組み。               |

---

**凡例:**

- `[実装済み]`: 実際のアプリで画面があり、操作可能な状態。
- `[実装済み(UI)]`: 画面構成はあるが、複雑なロジックを調整中のプロトタイプ。
- `[計画中]`: 要件定義書に記載されており、今後追加される予定の機能。
