# 機能一覧（現在の実装状況）

本サービス「masakinihirota」で現在利用可能な機能と、今後実装予定の機能の一覧です。

## ログイン・アカウント

- **[実装済み] ログイン機能**: Googleアカウント、GitHubアカウントで安全にログインできます。Auth.jsを導入し、セキュリティを向上させました。
- **[実装済み] ルートアカウント**: ログインすると、あなた専用の「ルートアカウント」が作成されます。

## プロフィール（千の仮面）

- **[実装済み] プロフィール作成**: 活動目的に合わせて、複数のプロフィール（仮面）を使い分けることができます。
- **[実装済み] リーダー/メンバー選択**: プロフィールごとに、自分でグループを作る「リーダー」か、既存のグループに参加する「メンバー」かを選べます。

## 経済・報酬 (Phase 4: Economy)

- **[実装済み] マーケットプレイス**: ユーザー間でアイテムやサービスの取引ができます。Glassmorphismデザインを採用し、高級感のあるUIで体験できます。
- **[実装済み] リワードシステム**: 毎日のログインや活動に応じてポイントを獲得できます。
- **[実装済み] ポイント管理**: 獲得したポイントの履歴確認や、サービス内での決済に利用できます。

## マッチング・コミュニティ (Phase 5: Matching)

- **[実装済み] マッチングハブ**: Tinder風のUIで、自分に合う仲間を直感的に探すことができます。
- **[実装済み] マンダラチャート**: 目標達成やスキルの可視化を行うためのツールです。
- **[実装済み] 基本マッチングロジック**: タグの一致とベクトル類似度（セマンティック検索）を組み合わせた高度なスコアリングを開始しました。

## その他

- **[実装済み] 開発ダッシュボード**: プロジェクトの進捗や、各機能のステータスをバッジ形式で確認できます。
- **[計⁠画中] メンバ⁠ー段階解放**: 活動に応じて、使える機能が少しずつ増えていく仕組みです。
- **[計画中] AIによる詳細マッチング**: AIを活用して、より深い価値観の一致を分析し、最適なパートナーや仲間を提案する高度な機能です。
