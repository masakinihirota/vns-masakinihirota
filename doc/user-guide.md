# 🔰 ユーザーガイド

VNS masakinihirota へようこそ！
ここでは、現在利用できる主な機能の使い方を紹介します。

## 🚀 はじめの一歩

### 1. ログイン・登録
トップページの「ログイン」ボタンから開始します。
- **Google / GitHub**: お持ちのアカウントですぐに始められます。
- **匿名**: アカウントなしでまずは雰囲気を体験できます（機能に制限があります）。

### 2. プロフィールの作成
ログイン後、「ユーザープロフィール」を作成しましょう。
- **名前**: ニックネームでOKです。
- **目的**: 何のために使いますか？（趣味仲間探し、仕事、雑談など）
- **役割**: グループを作りたいなら「リーダー」、参加したいなら「メンバー」を選びます。

💡 **ポイント**: プロフィールは複数作れます。「仕事用の真面目な顔」と「趣味全開の顔」を使い分けましょう。

## 💖 自分の「好き」を登録する

マッチングの精度を上げるために、あなたの価値観や好きな作品を教えてください。

### 価値観の登録
- 用意された質問に答えることで、あなたの考え方をシステムが理解します。
- 正直に答えるのが、良い出会いのコツです。

### 作品の登録
- **作品リスト**ページへ移動します。
- 好きな映画、漫画、ゲームなどを検索して登録します。
- **Tier評価**: その作品がどれくらい好きか、3段階で評価します。
  - **Tier 1**: 人生最高の作品！神！
  - **Tier 2**: 大好き、おすすめできる。
  - **Tier 3**: 普通に好き、悪くない。

## 🤝 仲間を見つける

### 自動マッチング
- 1日1回、あなたと価値観が近い人が自動で紹介されます。
- 気になったら「ウォッチ」してみましょう。

### 手動マッチング
- 条件を指定して検索できます。
- 特定の作品が好きな人、同じスキルを持つ人などを探せます。

### 組織（グループ）と国
- **グループに参加**: 気になるグループに「参加申請」を出してみましょう。
- **グループを作る**: 自分でリーダーになって、理想の居場所を作ることもできます。
- **国へ行く**: グループが集まる大きなコミュニティ「国」を覗いてみましょう。
