# 操作ガイド

現在実装されている機能の使い方の説明です。

## 1. ログイン・会員登録

右上のログインボタンから、GoogleやGitHubのアカウントを使って登録できます。
※現在、匿名ログイン（お試し体験）はメンテナンス中のため、SNSアカウントでのログインをお願いいたします。

## 2. プロフィール（仮面）を作る

「千の仮面」システムにより、あなたの多面性を表現できます。

- 1つのアカウントで、仕事用、趣味用、悩み相談用など、目的別に複数のプロフィールを作成できます。
- 自分で活動を主導したい場合は「リーダー」、既存のグループを応援したい場合は「メンバー」を選びます。

## 3. 好きな作品を登録・評価する

あなたの「価値観」を可視化するために、好きな作品を登録しましょう。

- 作品を検索し、自分のプロフィールに追加します。
- Tier 1（非常に大切）〜 Tier 3（好き）の3段階で評価することで、似た価値観を持つ人が見つかりやすくなります。

## 4. グループに参加・建国する

- **グループを探す**: 「グループ一覧」から、気になる活動をしている場所を探せます。
- **グループを作る**: リーダーとしてプロフィールを作成すると、即座に自分のグループを持つことができます。
- **国（Nation）**: 複数のグループが所属する大規模なコミュニティに参加し、より広い交流を楽しめます。

## 5. ポイントと信頼

- 毎日のログインや活動でポイントが貯まります。
- 「オアシス宣言」を守り、優しい世界を作ることで「信頼日数」が積み重なり、より高度な機能が使えるようになります。
