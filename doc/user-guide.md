# 操作ガイド

現在実装されている機能の使い方の説明です。

## 1. サービスを体験する
トップページの「お試し体験（Trial）」ボタンから、匿名でサービスを体験できます。

## 2. ログインする
右上のログインボタンから、GoogleやGitHubのアカウントを使って登録できます。

## 3. プロフィールを作る
「千の仮面」システムにより、あなたの多面性を表現できます。
- 仕事用のプロフィール、趣味用のプロフィールなど、目的別に作成してください。
- 自分で活動を始めたい場合は「リーダー」、誰かの活動に参加したい場合は「メンバー」を選びます。

## 4. 好きな作品を登録・評価する
あなたの価値観を仲間に伝えるために、好きな作品を登録しましょう。
- 作品を検索、または新しく登録します。
- その作品があなたにとってどれくらい大切か（Tier 1〜3）を設定します。

## 5. グループを探す
「グループ一覧」から、気になる活動をしているグループを探すことができます。
