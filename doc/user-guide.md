# VNS masakinihirota 利用者ガイド

VNS masakinihirota へようこそ！このガイドでは、現在実装されている機能を使って、どのように「価値観での繋がり」を楽しめるかを説明します。

## 1. 準備：自分の「盾（プロフィール）」を作る

VNSでは、一人の人間が複数の顔を持つことを「千の仮面」と呼びます。

- **操作**: 右上のアイコンまたは「プロフィール管理」から、新しいプロフィールを作成しましょう。
- **ヒント**: 「エンジニアとしての自分」「アニメ好きな自分」など、目的別に分けるのがコツです。
- **お試し**: ユーザー登録前でも「お試し体験」でプロフィールの作成を体験できます（データはブラウザにのみ保存されます）。

## 2. 自分の「好き」を伝える：作品登録と評価

あなたのプロフィールを豊かにするために、好きな作品を登録しましょう。

- **操作**: 「作品登録」メニューから作品を検索または登録し、**Tier (ティア)** を設定します。
- **Tierの意味**:
  - `Tier 1`: 人生を変えた、最高の作品
  - `Tier 2`: とても好きで、よく話題にする作品
  - `Tier 3`: 好きな作品の一つ

## 3. 自分の考えを登録する：価値観回答

プロフィールに価値観を紐付けます。

- **操作**: 「価値観アンケート」からお題を選んで回答してください。
- **効果**: ここで登録したデータが、後で「自動マッチング」の重要な手がかりになります。

## 4. 繋がりを探す：マッチング

準備ができたら、自分と感性が近い人を探してみましょう。

- **操作**: 「マッチング」メニューを選択し、使用する「仮面」を選んでから、「自動」または「手動」マッチングを開始します。
- **次のステップ**: 気になる相手を見つけたら、まずは「ウォッチ（見守り）」から始めてみましょう（※現在、組織参加機能は開発中です）。

## 5. 自分を深める：マンダラチャート

自分の目標や、スキルの整理に使えます。

- **操作**: 「マンダラチャート」から、中央に達成したい目標を書き、周りにそれを支える要素を埋めていきましょう。
