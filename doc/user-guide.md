# 操作ガイド

現在実装されている機能の使い方の説明です。

## 1. ログイン・会員登録

右上のログインボタンから、GoogleやGitHubのアカウントを使って登録できます。
※現在、匿名ログイン（お試し体験）はメンテナンス中のため、SNSアカウントでのログインをお願いいたします。

## 2. プロフィール（仮面）を作る

「千の仮面」システムにより、あなたの多面性を表現できます。

- 1つのアカウントで、仕事用、趣味用、悩み相談用など、目的別に複数のプロフィールを作成できます。「1000の仮面」メニューから、5つのステップで作成します。
- 自分で活動を主導したい場合は「リーダー」、既存のグループを応援したい場合は「メンバー」を選びます。

## 3. あなたの「繋がり」を可視化する

「Identity」メニューから、あなたの現在のプロフィール構成や、他者との関係性をグラフ形式で確認できます。どの仮面がどのコミュニティに属しているかを一目で把握できます。

## 3. 好きな作品を登録・評価する

あなたの「価値観」を可視化するために、好きな作品を登録しましょう。

- 作品を検索し、自分のプロフィールに追加します。
- Tier 1（非常に大切）〜 Tier 3（好き）の3段階で評価することで、似た価値観を持つ人が見つかりやすくなります。

## 4. マッチングハブで仲間を探す

「マッチングハブ」では、あなたの価値観（作品評価やマンダラチャート）に基づいたおすすめユーザーが表示されます。

- 気になるユーザーを「ウォッチ」したり、共通の「好きな作品」を確認したりして、つながりのきっかけを作ります。

## 5. マーケットでアイテムを取引する

ポイントを使って、作品の布教やスキルの提供などの取引が可能です。

- ガラスのような美しいデザインのカードで、出品内容や価格を確認できます。
- 出品者が自分自身の価値観に近いかどうかを、プロフィールアイコンから瞬時に判断できます。

## 6. ポイントと報酬（リワード）

- **デイリーボーナス**: 毎日ログインして「リワード」画面からポイントを受け取ることができます。
- **信頼日数**: 「オアシス宣言」を守り、優しい世界を作ることで積み重なります。信頼が高まると、マーケットでの高額取引や、特別なグループへの参加が解禁されます。

## 7. グループに参加・建国する

- **グループを探す**: 「グループ一覧」から、気になる活動をしている場所を探せます。
- **グループを作る**: リーダーとしてプロフィールを作成すると、即座に自分のグループを持つことができます。
- **国（Nation）**: 複数のグループが所属する大規模なコミュニティに参加し、より広い交流を楽しめます。
