# good-life

メーガン・サリバン教授（ノートルダム大学）の TED トーク「良い人生を送るための最も重要な美徳」に基づいた、愛、弱さ、そして善き人生についてのコンテンツを表示するコンポーネントです。

## 概要

このコンポーネントは、TED トークの内容を「対話レポート」と「全文書き起こし（日本語・英語）」の2つの形式で提供します。

- **対話レポート**: ヒロタと真咲の二人が、トークの内容についてカジュアルに語り合う形式で、要点を分かりやすくまとめています。
- **全文書き起こし**: トークの全内容を、言語切り替え機能付きで提供します。

## 出典

- **タイトル**: The Most Important Virtue for a Good Life
- **スピーカー**: Meghan Sullivan
- **URL**: [TED Talk - YouTube](https://www.youtube.com/watch?v=g83Xl4-mvF8)

- `good-life.tsx`: メインコンポーネント
- `good-life.test.tsx`: テスト（レンダリングおよびアクセシビリティ）
- `index.ts`: バレルエクスポート
