# Onboarding Trial Component

## 概要
体験版ユーザーが VNS に参加するためのオンボーディングフローを提供します。

## 構成
- `onboarding-trial-form/`: 登録フォーム本体。
  - `OnboardingTrialForm.tsx`: メインのマルチステップフォーム（全6ステップ）。
- `choice/`: 登録完了後の参加パス選択画面。
- `steps/`: 体験版専用のステップコンポーネント。
- `ui/`: 専用のUIコンポーネント（サイドバー等）。

## 登録フロー (6ステップ)
1. **オアシス宣言**: 利用規約への同意。
2. **活動拠点**: 既存の `onboarding-pc` コンポーネントを再利用。
3. **活動時間**: 既存の `onboarding-pc` コンポーネントを再利用。
4. **アイデンティティ**: 星座と世代の設定（体験版専用）。
5. **使用言語**: 既存の `onboarding-pc` コンポーネントを再利用。
6. **確認**: 設定内容の確認と登録。

## データ管理
登録されたデータは `src/lib/trial-storage.ts` を通じて `localStorage` に保存されます。

## 注意事項
- デザインシステム（Glassmorphism）に準拠しています。
- 既存の `onboarding-pc` コンポーネントを再利用しているため、型定義の不整合に注意してください（`as any` によるキャストで調整しています）。
