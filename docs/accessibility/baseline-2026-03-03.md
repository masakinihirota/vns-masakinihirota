## ⚠️ 問題・未実装（Issues）

### Priority 1（緊急・Phase 2で対応必須）

#### 1. ナビゲーションが `<nav>` で包まれていない
**場所**: `src/components/layout/header/header.tsx`
**問題**:
```tsx
❌ ヘッダーのアクションエリアが <nav> タグなし
  <div className="flex items-center gap-4">
    <AdsToggle />
    <LanguageToggle />
    <ThemeToggle />
  </div>
```

**修正案**:
```tsx
✅ <nav aria-label="ページコントロール"> で包む
  <nav aria-label="ページコントロール" className="flex items-center gap-4">
    <AdsToggle />
    <LanguageToggle />
    <ThemeToggle />
  </nav>
```

**影響度**: 中（ランドマークナビゲーション）
**修正難度**: 低

---

#### 2. トグルボタンに aria-pressed/aria-label がない
**推定場所**:
- AdsToggle（広告のON/OFF）
- ThemeToggle（ライト/ダークモード）
- LanguageToggle（言語切替）

**問題**: スクリーンリーダーが状態を読み上げられない

**修正案**: 各にトグルボタンに以下を追加
```tsx
<button
  aria-pressed={isAdvertising}
  aria-label="広告を表示"
  onClick={() => setAdvertising(!advertising)}
>
  Ad Icon
</button>
```

**影響度**: high（スクリーンリーダー互換性）
**修正難度**: 低（コンポーネント内修正）

---

#### 3. ランディングページのリンクスタイリング
**場所**: `src/app/page.tsx` - CTAボタン
**問題**:
```tsx
<Link href="/login" className="...">
  ログイン
</Link>
```
セマンティック上、`<Link>` は OK ですが、ボタンスタイルの場合 `role="button"` 検討

**修正案**: Link のアクセシビリティは OK（Next.js自動処理）

**影響度**: 低
**修正難度**: 低

#### 4. 見出し階層確認
**場所**: `src/app/page.tsx`
**問題**:
