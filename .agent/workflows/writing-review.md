# Writing Review Workflow

このワークフローは、`writing-style` スキルに基づいてドキュメントの文体や表現をチェックします。

## 手順

1.  **ルールの読み込み**:
    - `.agent/skills/writing-style/SKILL.md` を読み込み、執筆基準を確認します。

2.  **対象ファイルの特定**:
    - ユーザーが指定したファイル、または現在開いているファイルを対象とします。

3.  **文体チェックの実行**:
    - `writing-style` スキルの各項目（文体、構成、表記、フォーマット）に沿って、対象ファイルをレビューします。
    - 特に「ですます調の統一」や「補助動詞のひらがな化」に注意してチェックします。

4.  **改善案の提示**:
    - 修正が必要な箇所を具体的に指摘し、改善後の文案を提示します。

## コマンド例

- `/writing-review`: 指定したファイルの文体チェックを実行
