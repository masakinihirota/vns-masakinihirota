### 4. ARIA 属性の適切な使用
- [ ] 4.1 `role` 属性（正しい役割を明示）
  - ❌ 誤: `<div role="button">` → ✅ 正: `<button>`

- [ ] 4.2 `aria-label` / `aria-labelledby` / `aria-describedby`
  - [ ] アイコンボタン: `<button aria-label="メール送信"><Icon /></button>`
  - [ ] フォーム エラー: `<input aria-describedby="error-msg" />`
- [ ] 4.3 `aria-hidden="true"` で装飾要素を除外
  - [ ] スクリーンリーダーから「不可視」
- [ ] 4.4 `aria-live="polite"` / `aria-live="assertive"` でリアルタイム更新通知
  - [ ] トースト通知: `aria-live="assertive"`
  - [ ] ローディング状態: `aria-live="polite"`
- [ ] 4.5 `aria-expanded`, `aria-pressed`, `aria-selected` で状態表示
  - [ ] メニュー: `aria-expanded="false"` → `aria-expanded="true"`
  - [ ] チェックボックス: `aria-checked="true"`

**完了条件**
- [ ] ARIA 属性が過剰・不足なく正確
- [ ] Wave（アクセシビリティ監査ツール）で ARIA エラーゼロ
