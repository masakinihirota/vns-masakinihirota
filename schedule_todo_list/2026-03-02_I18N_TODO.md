### 4. エラーメッセージの国際化
- [ ] 4.1 `/src/lib/errors/messages.ts` を翻訳キー型に変更
  ```typescript
  // 変更前
  export const ERROR_MESSAGES = {
    'E001_AUTH_REQUIRED': 'ログインが必要です。',
  };

  // 変更後
  export const ERROR_MESSAGE_KEYS = {
    'E001_AUTH_REQUIRED': 'errors.auth_required',
    'E002_PERMISSION_DENIED': 'errors.permission_denied',
  };
  ```
- [ ] 4.2 UI コンポーネントで `useTranslations()` 使用
