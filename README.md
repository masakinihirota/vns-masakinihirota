# VNS masakinihirota

> **昨日僕が感動した作品を、今日の君はまだ知らない。価値観でつながるネットワークサービス。**

VNS (Value Network Service) は、単なる作品評価サイトではなく、個人の「価値観」を軸にした新しいマッチング体験を提供します。

## 主要なコンセプト

- **ゲスト閲覧モード**: プロフィール登録前の状態では、全ユーザーの評価トレンドを人気ランキング形式で見て回ることができます。
- **プロフィールの登録**: 自分の好きな作品や感性を登録することで、自分に似た感性を持つ人々との繋がりや、おすすめの作品が見つかるようになります。
- **Tier 1, 2, 3**: 作品の人気や評価を、単なる数字ではなく、熱量に応じた3つの階層構造で分かりやすく表現します。

## セットアップ

### データの初期登録

提供されたアニメ・漫画タイトルをカタログ登録するには：

1. アプリにログインしていることを確認してください。
2. 開発サーバーを起動し、ブラウザで以下にアクセスしてください：
   `http://localhost:3000/api/admin/seed-works`
3. JSONレスポンスが表示されるまで待ちます。
4. `/works` にアクセスして、データが反映されたことを確認してください。
